Detailed Description

template<>
class atmi::Tfield< char * >

Specialization of template Tfield which handles CARRAY typed fields.

This class is using a std::string object to hold and handle char * data When a new instance is created, occurence is set to 0 making it possible to set values withour a prior call to add.
