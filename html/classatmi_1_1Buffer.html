Detailed Description

FML Buffer

A fielded buffer is composed of Fields of fixed length (for example, TField<long>, TField<short>), and field for varying length fields (for example TField<string>).

void atmi::Buffer::set_buffer ( FBFR32 *  b)

Set a new buffer reference. Previous buffer is deallocated before assign the passed value.

Parameters
bset FML32 buffer reference

replaces the current buffer reference by the given one.

If preveous reference was allocated by this instance, then the buffer is freed before setting the new buffer reference. The buffer instance is then flagged as not allocated. Meaning that it's up to you to free the memory that was allocated.

Parameters
breference of an FML buffer.
Exceptions
Exceptionif the buffer is not an FML buffer.
