void atmi::Queue::set_new_corrid ( )

Calculate and set qctl corid

Correlation ID is made of pid+threadid+<current time="" in="" milliseconds>="">. When the correlation is set the flags TPQGETBYCORRID and TPQCORRID are also set. This means that subsequent calls to dequeue methods will get only message with the correleation id that was last set.
